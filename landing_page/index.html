Description

Little Holby is a discord administrative assistant. It's meant to make the discord experience smoother between the staff and the student (tutors). The main feature we have for know is creating tickets and the rest of the commands are to manage users such as mute, ban, kick, etc. All the commands that we have available are in the header section above.

Bot Functions

!ticket

This is a visual example of what the user will see when they use the !ticket command on the server. They will see a green icon to open a ticket and when they click on it, it will open a new tab asking for what is the purpose of the ticket.

!kick

This is a visual example of what will appear when the user use !kick. It will ask to name an @username in the current server. You can also put any reason why is the action is being executed but not necessary. One the command is executed with the @username, it will be out of the server.
